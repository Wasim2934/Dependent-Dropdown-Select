# Dependent Dropdown Select (React + Tailwind CSS)

A clean and responsive **Dependent Dropdown Select** project built using **React.js** and **Tailwind CSS**.

This project demonstrates how to create a **country → city dependent dropdown**, where the city options change dynamically based on the selected country.

---

## Live Demo - https://wasim2934.github.io/Dependent-Dropdown-Select/


## 🚀 Features

- Select a **Country**
- Dynamically show related **Cities**
- Reset city when country changes
- Disabled city dropdown until country is selected
- Clean and modern UI using **Tailwind CSS**
- Beginner-friendly React state management project

---

## 🛠️ Tech Stack

- **React.js**
- **Tailwind CSS**
- **JavaScript (ES6)**
- **JSX**
